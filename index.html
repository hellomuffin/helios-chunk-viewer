Helios — Autoregressive Chunk Viewer

Each task below is a single Helios-Base (50-step) run given a list of step instructions — one step = one chunk. The model generates the video chunk by chunk (81 frames / ~3.4s each, num_latent_frames_per_chunk=21), every chunk conditioned on the previous via Unified History Injection, and each chunk is driven by exactly one step prompt (no interpolation). For each prompt-set you see the full stitched video and the individual chunks with the instruction that drove each one. Note: with one-step-one-chunk and no prompt-blending, each chunk boundary is a hard instruction switch, so visible transitions there are the model changing scene on command (the authors' built-in smoothing — prompt interpolation — is intentionally disabled here).
